## Overview
**Zelda: Twilight Princess PC** is a native PC port of the beloved classic. Experience the dark and epic story, unique wolf transformation mechanics, vast world of Hyrule, challenging dungeons, and cinematic gameplay with improved performance, higher resolutions, and modern controls.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run the game executable as Administrator
3. Adjust graphics settings according to your PC
4. Enjoy the adventure in Hyrule

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Game doesn't start         | Run as Administrator                          |
| Low FPS                    | Lower resolution or graphics settings         |
| Black screen               | Switch to Borderless Windowed mode            |
| Antivirus warning          | Add the folder to exclusions                  |
| Controller issues          | Check controller settings in game             |
